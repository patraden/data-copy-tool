class SparkPartitionReaderSourceShapeGraph extends GraphStage[SourceShape[Row]]

A GraphStage represents a reusable graph stream processing operator defined for SparkPartitionReader. Normally a GraphStage consists of a Shape which describes its input and output ports and a factory function that creates a GraphStageLogic which implements the processing logic that ties the ports together. This class specifically implements a SourceShape with single output and no inputs.
