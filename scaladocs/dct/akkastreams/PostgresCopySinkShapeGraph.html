Companion object PostgresCopySinkShapeGraph

class PostgresCopySinkShapeGraph extends GraphStageWithMaterializedValue[SinkShape[ByteString], Future[Long]]

A GraphStage represents a reusable graph stream processing operator defined for Postgres COPY command. Normally a GraphStage consists of a Shape which describes its input and output ports and a factory function that creates a GraphStageLogic which implements the processing logic that ties the ports together. This class specifically implements a SinkShape with single input and no outputs.
